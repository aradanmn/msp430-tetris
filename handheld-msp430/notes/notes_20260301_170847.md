## KiCad 9 Format Specification (Researched)

**Sources**: KiCad sch_file_versions.h (via web search), KiCad forum posts, official release notes

### Version history (from sch_file_versions.h)
| Version | Change |
|---------|--------|
| 20231120 | `generator_version` field introduced (KiCad 8.0 base) |
| 20241004 | `hide` changed from `(hide yes)` to bare boolean `(hide)` |
| 20241209 | Private flags for SCH_FIELDs |
| 20250114 | Full paths for text variable cross references (KiCad 9.0) |

### KiCad 9 Schematic Header
```scheme
(kicad_sch
  (version 20250114)
  (generator "eeschema")
  (generator_version "9.0")
  (uuid "...")
  (paper "B")
  (title_block ...)
  (lib_symbols ...)    ← MUST be inside kicad_sch, after title_block
  ;; placed symbols, wires, labels, junctions, no_connects ...
  (sheet_instances     ← MUST be at the very end
    (path "/" (page "1"))
  )
)
```

### KiCad 9 Placed Symbol
```scheme
(symbol (lib_id "SYMBOL_NAME") (at cx cy rot)
  (unit 1)
  (exclude_from_sim no)   ← NEW in KiCad 9
  (in_bom yes)
  (on_board yes)
  (dnp no)                ← NEW in KiCad 9
  (uuid "...")
  (property "Reference" "U1"
    (at cx cy 0) (effects (font (size 1.27 1.27)) (hide)))
  (property "Value" "MSP430G2553"
    (at cx cy 0) (effects (font (size 1.27 1.27)) (hide))))
```

### KiCad 9 Net Label
```scheme
(label "NET_NAME" (at x y rot)
  (fields_autoplaced yes)
  (effects (font (size 1.27 1.27)) (justify left))   ← always left, rot controls direction
  (uuid "..."))
```

### KiCad 9 Property Hide (format 20241004+)
```scheme
;; KiCad 8: (hide yes)
;; KiCad 9: bare boolean token
(effects (font (size 1.27 1.27)) (hide))   ← just (hide), not (hide yes)
```

### KiCad 9 Power Symbol
```scheme
(symbol (lib_id "VCC") (at x y 0)
  (unit 1) (exclude_from_sim no) (in_bom yes) (on_board yes) (dnp no)
  (uuid "...")
  (property "Reference" "#PWR" (at x y 0) (effects (font (size 1.27 1.27)) (hide)))
  (property "Value" "VCC" (at x y 0) (effects (font (size 1.27 1.27)) (hide))))
```

### Coordinate System
```
Symbol local: Y increases UP (math coords)
Schematic world: Y increases DOWN (screen coords)
Formula: world = (cx + lx,  cy - ly)
```

### Pin Endpoint Offsets (for ic_sym with body_hw=5.08)
- Left side: `lx = -(body_hw + 2.54) = -7.62`, stub angle = 0
- Right side: `lx = +(body_hw + 2.54) = +7.62`, stub angle = 180
- Pitch: 2.54mm, centred → top pin ly = (n-1)×1.27

### Wire Stub Convention
Every label has a 2.54mm wire stub FROM the pin endpoint:
- rot=0   (right-side pin): wire → (+2.54, 0), label at far end
- rot=180 (left-side pin):  wire → (−2.54, 0), label at far end
- rot=270 (top pin):        wire → (0, −2.54), label above
- rot=90  (bottom pin):     wire → (0, +2.54), label below
